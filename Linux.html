Basic linux commands:


ls - list files

cd - change directory (go back a directory: 'cd ..')
pwd - print working directory

mv - move a file
cp - copy a file
rm - remove a file

mkdir - make a directory
rmdir - remove a directory (must be empty)

clc - clear window

gedit - basic text editor

whoami - print account name
df - check file system
sleep X - pause X seconds
